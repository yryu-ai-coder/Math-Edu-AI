📖 1. What is a Function?

A function is a relationship where each input (x) has exactly one output (y).

🔢 Example: Temperature Conversion

Converting Celsius to Fahrenheit: F = (9/5)C + 32

For each Celsius temperature, there's exactly one Fahrenheit temperature.

Celsius (C) Fahrenheit (F)
0°C 32°F
10°C 50°F
20°C 68°F
30°C 86°F

📈 2. Linear Functions

Linear functions have the form f(x) = mx + b, where m is the slope and b is the y-intercept.

💰 Example: Salary Calculation

If you earn $30,000 base salary plus $2,000 commission per sale:

S(x) = 2000x + 30000

Where x is the number of sales and S(x) is the total salary.

📊 3. Function Notation

Functions can be written as f(x) = expression, where f is the function name and x is the input.

📝 Examples:

  • f(x) = 2x + 3 - Linear function
  • g(x) = x² - Quadratic function
  • h(x) = 2^x - Exponential function

To find f(5) in f(x) = 2x + 3: f(5) = 2(5) + 3 = 13

📈 4. Exponential Functions

Exponential functions have the form f(x) = a·b^x, where a is the initial value and b is the growth factor.

🦠 Example: Bacterial Growth

If bacteria double every hour starting with 100 bacteria:

N(t) = 100 × 2^t

Where t is time in hours and N(t) is the number of bacteria.

Time (hours) Bacteria Count
0 100
1 200
2 400
3 800

✏️ Practice Problems

Problem 1: Function Evaluation

Given f(x) = 3x - 2, find f(4) and f(-1).

Answer: f(4) = 10, f(-1) = -5

Explanation: f(4) = 3(4) - 2 = 12 - 2 = 10

f(-1) = 3(-1) - 2 = -3 - 2 = -5

Problem 2: Linear Function from Points

Find the linear function that passes through points (2, 5) and (4, 9).

Answer: f(x) = 2x + 1

Explanation: Slope = (9-5)/(4-2) = 4/2 = 2

Using point (2,5): 5 = 2(2) + b → 5 = 4 + b → b = 1

So f(x) = 2x + 1

Problem 3: Exponential Function

If a population starts with 500 and grows by 20% each year, find the function P(t).

Answer: P(t) = 500 × (1.2)^t

Explanation: 20% growth means the growth factor is 1.2 (100% + 20% = 120% = 1.2)

Initial value is 500, so P(t) = 500 × (1.2)^t
